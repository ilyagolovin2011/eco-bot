Экологический бот.

Вот оригинальная версия бота - @eco_logy_bot

Бот о глобальном потеплении и их решениях.

За основу проекта взят телеграм-бот, который выдает информацию о глобальном потеплении и предлагает советы по улучшению экологической ситуации в направлении глобального потепления. Еще он показывает методы улучшения в сфере глобального потепления.

Функции: в боте есть много разных функций
1. Красивые кнопки для удобного использования бота.
2. Краткая информация о глобальном потепление, а так же прикрепленна ссылка на подробную информацию.
3. Так же дана краткая информация о актуальных данных на данный момент с прикрепленной ссылкой на подробную информацию.
4. Присутствует не большой тест "Моё влияние", где вы будете отвечать на вопросы связанные с вашей жизнью о том как вы пользуетесь разными вещами. В конце покажет информацию о том как вы влияете на окружающею среду и как это можно исправить или улучшить.
5. Так же там присутствует информация о путях к улучшению.
6. Если у вас появятся вопросы или баги с ботом, то там есть информация о том куда написать в случай проблемы

Вот подробнная информация как установить бота под своё использование
1. Установить python.
2. Использовать приложения IDLE, я использую Visual Studio Code.
3. Загрузите фотографии и файл с ботом bot.py в папку где будет храниться ваш бот
4. Создайте файл config.py чтобы там хранился токен для бота, поместите туда переменную token1 = 'ваш токен' (ваш токен) замените на ваш действующий токен бота, его получить можно в боте телеграмм @BotFather. После того как вы вставили токен он будет примерно выглядить так token1 = '15212521:ksasafafs'
5. Установите нужные библеотеки для работы бота
pip install pyTelegramBotAPI  # для telebot
pip install requests          # для requests
6. После того как вы все установили бот будет работать без ошибок.
